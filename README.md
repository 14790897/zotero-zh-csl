# GB/T 7714—2015（顺序编码，双语）去除OL

## 简介

本样式基于 [GB/T 7714—2015（顺序编码，双语）](https://zotero-chinese.com/styles/GB-T-7714—2015（顺序编码，双语）) 修改而来，原样式由牛耕田和 Zeping Lee 维护。
点击 `GB-T-7714—2015（顺序编码，双语）去除OL.csl` 查看具体内容，并可以下载。

## 与原版的主要区别

### 1. 文献载体标识 OL 按类型判断

原版对所有含有 URL 或 DOI 的文献一律添加 `OL` 载体标识，例如：

```
张三. 论文题目[J/OL]. 期刊名, 2024...
```

**本版修改为仅对以下类型自动添加 `OL`：**

| 类型标识 | 对应文献类型 |
|---------|------------|
| `A` | 档案、预印本（article, bill, collection 等） |
| `EB` | 网络文献（post, post-weblog, webpage） |

其余文献类型（J、M、C、D、P、R 等）即使含有 URL/DOI，也**不添加** `OL` 标识。

### 2. URL 和 DOI 链接按类型判断显示

原版对所有文献均在末尾显示 URL 和 DOI。

**本版修改为仅对 A 类和 EB 类文献显示 URL 和 DOI**，其他类型不显示。

### 3. 访问日期按类型判断显示，并提供默认日期

原版对所有含 URL/DOI 的文献显示访问日期。

**本版修改逻辑如下：**

- 仅对 **A 类和 EB 类**文献显示访问日期
- 如果该条目**已填写访问日期**，则显示实际日期
- 如果该条目**未填写访问日期**，则自动填充默认日期 `[2026-01-01]`

## 效果对比

| 文献类型 | OL标识 | URL/DOI | 访问日期 |
|---------|-------|---------|--------|
| 期刊（J） | ✗ | ✗ | ✗ |
| 专著（M） | ✗ | ✗ | ✗ |
| 学位论文（D） | ✗ | ✗ | ✗ |
| 会议论文（C） | ✗ | ✗ | ✗ |
| 预印本/档案（A） | ✓ | ✓ | ✓（无日期时显示2026-01-01）|
| 网络文献（EB） | ✓ | ✓ | ✓（无日期时显示2026-01-01）|

## 安装方法

1. 下载 `GB-T-7714—2015（顺序编码，双语）去除OL.csl` 文件
2. 打开 Zotero → **编辑 → 首选项 → 引用 → 样式**
3. 点击 **+** 按钮，选择下载的 `.csl` 文件导入
